# બેકએન્ડ ઘટકો અને કાર્યક્ષમતા (Backend Components and Functions)

આ પ્રોજેક્ટના બેકએન્ડમાં મુખ્ય ત્રણ ભાગો છે: **API સર્વર**, **ડિસ્કોર્ડ બોટ**, અને **ડેટાબેઝ**. નીચે દરેક ઘટક શું કામ કરે છે તેની વિગતવાર સમજૂતી આપવામાં આવી છે.

---

## ૧. API સર્વર (Express API Server)
આ સર્વર ડેશબોર્ડ (Frontend) અને ડેટાબેઝ વચ્ચે સેતુ તરીકે કામ કરે છે.

*   **Auth (ઓથેન્ટિકેશન):** `server/routes/auth.js` ફાઇલ ડિસ્કોર્ડ લોગિન (OAuth2) સંભાળે છે. તે યુઝરની ઓળખ ચકાસે છે જેથી તેઓ પોતાના સર્વરના સેટિંગ્સ બદલી શકે.
*   **API Routes:** `server/routes/api.js` માં બધા મુખ્ય એન્ડપોઇન્ટ્સ છે. તે સર્વરના સેટિંગ્સ (જેમ કે Auto-Translate, Bans, Text Replacements) મેળવવા અને અપડેટ કરવા માટે વપરાય છે.
*   **Middleware:** સુરક્ષા માટે અહીં CSRF પ્રોટેક્શન, CORS, અને Rate Limiting નો ઉપયોગ થયો છે જેથી સર્વર સુરક્ષિત રહે.
*   **Logging:** દરેક વિનંતી (Request) નો રેકોર્ડ રાખે છે જેથી ભૂલો શોધવામાં સરળતા રહે.

---

## ૨. ડિસ્કોર્ડ બોટ (Discord Bot)
આ બોટ ડિસ્કોર્ડ પર સીધી રીતે યુઝર્સ સાથે વાતચીત કરે છે અને અનુવાદનું કામ કરે છે.

*   **Events (ઘટનાઓ):** 
    *   `messageCreate`: જ્યારે કોઈ નવો મેસેજ આવે, ત્યારે તે ચેક કરે છે કે શું તેનો ઓટો-ટ્રાન્સલેટ કરવાનો છે.
    *   `messageReactionAdd`: જો કોઈ મેસેજ પર ઝંડા (Flag) ની પ્રતિક્રિયા આપે, તો તે મેસેજને તે દેશની ભાષામાં અનુવાદ કરે છે.
    *   `interactionCreate`: યુઝર જ્યારે સ્લેશ કમાન્ડ્સ (Slash Commands) વાપરે ત્યારે તેનો જવાબ આપે છે.
*   **Services:**
    *   `translationCore.js`: આ મેસેજને એક ભાષામાંથી બીજી ભાષામાં ફેરવવાનું મુખ્ય કામ કરે છે.
    *   `TTS Service`: મેસેજને અવાજમાં (Text-to-Speech) ફેરવવા માટે વપરાય છે.

---

## ૩. ડેટાબેઝ (MongoDB Models)
બધી માહિતી સાચવવા માટે MongoDB નો ઉપયોગ થાય છે. `server/models/` માં તેની સ્કીમા (Schema) વ્યાખ્યાયિત કરેલી છે.

*   **GuildSettings:** દરેક ડિસ્કોર્ડ સર્વરના સેટિંગ્સ સાચવે છે.
*   **AutoTranslateConfig:** કયા ચેનલમાં ઓટો-અનુવાદ ચાલુ છે તેની માહિતી.
*   **TranslateBan:** કયા યુઝર્સને અનુવાદ કરવા પર પ્રતિબંધ છે તેની યાદી.
*   **GuildStats:** બોટ દ્વારા કેટલા અનુવાદ થયા તેના આંકડા (Stats) સાચવે છે.
*   **TextReplacement:** અનુવાદ કરતી વખતે કયા શબ્દો બદલવા (Replace) તે નક્કી કરે છે.

---

## ૪. અન્ય મહત્વની ફાઈલો
*   `server/app.js`: API સર્વરનું મુખ્ય માળખું.
*   `src/index.js`: ડિસ્કોર્ડ બોટ શરૂ કરવાની મુખ્ય ફાઇલ.
*   `.env`: સિક્રેટ ટોકન્સ અને ડેટાબેઝ પાસવર્ડ સાચવવા માટે.

---

આ માળખું આખા પ્રોજેક્ટને વ્યવસ્થિત રીતે ચલાવવા માટે જવાબદાર છે.
